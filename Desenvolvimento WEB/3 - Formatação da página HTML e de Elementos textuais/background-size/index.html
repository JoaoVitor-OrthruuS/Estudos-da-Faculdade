Auto

valor padrão (mesma largura e altura da imagem);


Length

width height: define um valor em pixel para a largura e altura. Se apenas um é definido, o segundo fica como auto;


%

define um valor em porcentagem para a largura e altura. Se apenas um é definido, o segundo fica como auto;


cover

encaixa a imagem no fundo, preenchendo todo o espaço, sem repetição (pode ser que a imagem não apareça toda) ;


contain

encaixa a imagem perfeitamente no fundo da página;


initial

Valor padrão (mesma largura e altura da imagem);


Inherit

Herda a propriedade do elemento pai.
